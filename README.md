# Analisis-Konvergensi-Iterasi-Titik-Tetap
Mini project metode numerik menggunakan Metode Iterasi Titik Tetap (Fixed Point Iteration) untuk menyelesaikan persamaan nonlinear f(x)=x 3 −2x 2 +3x−1.  Dalam project ini dilakukan proses iterasi numerik menggunakan beberapa bentuk fungsi g(x) untuk menganalisis konvergensi menuju akar persamaan dengan bantuan Microsoft Excel.
